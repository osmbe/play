Play.OSM.be is a development playground for OpenStreetMap Belgium.

If you have a little project related to OpenStreetMap and need a place to let it live, you can do so here!

Content is managed at our Github Page

For now, the only content of this site is a few leaflet maps.

Wegenregister:
A map showing possible errors in OSM based on Wegenregister analysis.
The map is available here.
More info about the project here.

Historical map:
A map where you can overlay OpenStreetMap roads (and GPX tracks and Strava GPS) on top of historical maps of Flanders and Belgium ranging from 1740 to 1989. The map is here.
